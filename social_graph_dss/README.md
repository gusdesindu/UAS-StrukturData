# DSS Relasi Sosial Media

Sistem ini membantu pengguna menganalisis dan merekomendasikan relasi sosial media berbasis graph.

## Fitur
- Tambah user
- Tambah relasi
- Visualisasi graph
- Cari jalur pertemanan (BFS/Dijkstra)
- Rekomendasi teman (mutual friend)
- Analisis user populer

## Teknologi
- Python
- Streamlit
- NetworkX
- Matplotlib

## Cara Menjalankan
1. Install dependencies: `pip install -r requirements.txt`
2. Jalankan aplikasi: `streamlit run app.py`
